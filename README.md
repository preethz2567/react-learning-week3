# JS and React Learning
## What I Learned

### JavaScript Prerequisites (had to do these first)
- **Arrow functions** - no `return` keyword needed for single expressions
- **Destructuring** -objects and arrays. Similar to Python's tuple unpacking
- **Array methods** - `map`, `filter`, `find`, `reduce`. 
- JavaScript's `===` vs `==` — always use `===` (strict equality)

### React Basics
- **JSX** - it looks like HTML but it compiles to JavaScript. Rules: `className` not `class`, events are camelCase (`onClick`), must have one root element
- **useState** -hook that gives you a state variable and a setter. Always call the setter, never mutate state directly
- **Props** -data passed from parent to child. Like Java method parameters. Destructure them directly in the function signature
- **useEffect** - runs side effects. The dependency array controls when it re-runs. Empty `[]` = once on mount. `[variable]` = whenever that variable changes
- Component names MUST start with uppercase letter
